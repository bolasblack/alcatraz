    alca network-helper#

    Manage network helper for LAN access

    Synopsis#

    Manage the network helper for container LAN access.

    On macOS: Installs a helper container that runs nftables inside the container runtime VM for network isolation.

    On Linux: Configures nftables to include alcatraz rule files from /etc/nftables.d/alcatraz/ for persistent firewall rules.
